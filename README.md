# Guest House Management System

## Project Description
This is a Guest House Management System developed using Java, Spring Boot and MySQL. It manages guest records including check-in, check-out, room number and payment status.

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- Bootstrap

## Setup Instructions
1. Clone the repository.
2. Open the project in Eclipse.
3. Create a MySQL database named `guesthouse`.
4. Update the username and password in `application.properties`.
5. Run the Spring Boot application.
